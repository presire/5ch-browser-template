Ember — ガラス効果プレビュー

このページについて。
Ember のガラス効果のデザイン検討時に作成したモックアップを、ランディングから参照できる形にしたものです。設定 → 表示 → ガラス効果 で オフ / 軽量 / フル の3段階を切替できます。
実装版に対応するのは Variant 5 — Lite Glass（軽量モード）と Variant 4 — Fake Glass（フル）。Variant 1〜3 は採用されなかった検討案です。
右上のトグルでライト/ダーク切替可能。

実装方針 3 (フェイクガラス) の完成予想図。
✅ 両OS で確実に動作 — Tauri / WebView2 の透過に依存しない

軽量モードの完成予想図。
✅ macOS 体感負荷の大幅低減 — blur 半径を 6px に固定 + saturate 120% で GPU コンポジット負荷を最小化
✅ Windows ウィンドウ管理障害の回避 — 大面積 backdrop-filter を最小化することで WebView2 の再描画コストを抑制
❌ 奥行き感はほぼ失われる — 「すりガラス」→「半透明アクリル板」程度。Variant 4 の壁紙感は出ない
設定 UI 上は OFF / Lite / Full の3段階トグルとして提供する想定。dark variant の radial-gradient は 4層 → 1層 (top-left 紫のみ) まで削減。
推定負荷: フルガラス (現行 v0.0.134) を 100 とすると、軽量モードは 35〜45、フラット OFF は 5〜10。中間ポジションでガラス感の輪郭は維持。
