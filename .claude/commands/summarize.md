プロジェクトの最近の変更を要約してください。  

範囲: $ARGUMENTS  

引数の例:  
- `5` — 最新5コミット (デフォルト)
- `v0.0.8..HEAD` — 指定範囲
- `--since="2 weeks ago"` — 期間指定

以下を報告:  

1. **コミット一覧** — `git log --oneline` で対象コミットを表示  
2. **変更ファイル** — `git diff --stat` で変更されたファイルと行数  
3. **カテゴリ別要約**:  
   - 新機能
   - バグ修正
   - リファクタリング
   - テスト・CI
   - ドキュメント
4. **未コミット変更** — `git status` と `git diff --stat` で現在の作業状態
